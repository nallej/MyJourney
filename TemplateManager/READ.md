# TemplateManager 6
TemplateManager is a smaller version of TemplateBuilder.
TemplateManager do only one thing, creates a Template
### Recommendations
You should use Cloud Images with TemplateManager. 

# How to use
First you should read the scripts and if you trust the code, use it.

### Initial step on Proxmox
1. Go to any pool and create a empty folder
2. Go to that folder
3. Download the `dltm6` file: `wget https://github.com/nallej/MyJourney/raw/main/TemplateManager/dltm6`   
5. Edit the hidden `tm/.env` file
6. Run `bash dltm6`

### Run the main script on Proxmox
bash `tm6`
